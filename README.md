## Featured Projects

<details>
<summary><strong>OpsPilot — Autonomous Agent with Model Context Protocol Integration</strong></summary>
<br/>

OpsPilot is an autonomous operations agent built around an explicit **Plan–Execute–Evaluate** state machine, designed to safely automate multi-step operational workflows. It integrates the **Claude API** for reasoning and the **Model Context Protocol (MCP)** for dynamic, standardized tool discovery — exposing seven tools through a custom Node.js MCP server spanning operations and filesystem domains.

| | |
|---|---|
| **Stack** | Python, FastAPI, Claude API, Node.js (MCP Server), Redis, SQLite |
| **Scale** | 7 tools exposed via MCP across operations & filesystem domains |
| **Performance** | Explicit Plan–Execute–Evaluate state machine for reliable multi-step automation |
| **Security** | Human-in-the-loop approval workflows for high-risk operations |
| **Impact** | Standardized, interoperable tool discovery via Model Context Protocol |
| **Repository** | [github.com/ahan-jain/opspilot](https://github.com/ahan-jain/opspilot) |

The system prioritizes safety and observability: high-risk operations require explicit approval before execution, and a Redis-backed exponential backoff retry mechanism ensures resilience against transient failures. Every action is logged to a SQLite-backed audit trail, enabling complete traceability for debugging and post-hoc safety analysis — demonstrating how MCP can decouple agent reasoning from tool execution.

</details>

<details>
<summary><strong>InsightHub — AI-Powered Field Intelligence Platform</strong></summary>
<br/>

InsightHub is a full-stack, AI-powered field intelligence platform that brings real-time computer vision to safety and compliance inspections. At its core is a multi-model pipeline running five specialized YOLOv8 detectors — covering general objects, PPE, helmets, infrastructure, and fire — achieving an 85–90% detection rate across 10+ object classes.

| | |
|---|---|
| **Stack** | Next.js, FastAPI, Python, Ultralytics YOLOv8, Leaflet, ReportLab, IndexedDB |
| **Scale** | 5 specialized detectors across 10+ object classes |
| **Performance** | 85–90% detection rate for safety violations & infrastructure damage |
| **Security** | Offline-first PWA with local-first data persistence (zero-connectivity operations) |
| **Impact** | Automated compliance reporting and geo-tagged violation analytics |
| **Repository** | [github.com/ahan-jain/insighthub](https://github.com/ahan-jain/insighthub) |

Built with Next.js and FastAPI, the platform operates as an offline-first Progressive Web App backed by IndexedDB, allowing field teams to capture and process inspection data with zero connectivity and sync once back online. Detected violations are automatically geo-tagged via Leaflet maps, compiled into PDF compliance reports with ReportLab, and surfaced on an analytics dashboard tracking violation trends — turning raw detections into actionable compliance intelligence.

</details>
